# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it to us as soon as possible. We will take all security reports seriously.

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, please email us at [your-email@example.com](mailto:your-email@example.com).

Please include the following information in your report:

*   A clear and concise description of the vulnerability.
*   Steps to reproduce the vulnerability.
*   Any proof-of-concept code.

We will respond to your report within 48 hours.
